# CppChat
This project simulates a chat between two windows on the same computer. Each window acts as a separate user, allowing messages to be sent and displayed in real-time. The main purpose is to demonstrate communication between multiple windows within a single application using C++ and Qt.

## Requirements
- C++17 or higher
- Qt 6
- CMake or Qt Creator

## Run
1. Open the project in Qt Creator
2. Build the project
3. Run `main.cpp`
